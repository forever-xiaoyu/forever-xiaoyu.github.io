Markdown 语法

hexo 是使用 Markdown 编辑文章的，使用简洁，详见 Markdown 官方文档

注意事项

  • 使用 git clone 修改 hexo 主题时，如果是在 hexo 项目中，删掉主题中的 .git 文件
  • 使用 hexo new page 生成单页面
  • 修改语言时，依据当前主题的 languages 中的语言进行修改，每个主题都会有所差异
