canvasиғҪеҒҡдәӣд»Җд№Ҳпјҹ

жёёжҲҸ пјҡжёёжҲҸеңЁHTML5йўҶеҹҹе…·жңүдёҫи¶іиҪ»йҮҚзҡ„ең°дҪҚгҖӮHTML5еңЁеҹәдәҺWebзҡ„еӣҫеғҸжҳҫзӨәж–№йқўжҜ”FlashжӣҙеҠ з«ӢдҪ“гҖҒжӣҙеҠ зІҫе·§пјҢиҝҗз”ЁCanvasеҲ¶дҪңзҡ„еӣҫеғҸиғҪеӨҹд»ӨHTML5жёёжҲҸеңЁжөҒз•…еәҰе’Ңи·Ёе№іеҸ°ж–№йқўеҸ‘жҢҘжӣҙеӨ§зҡ„жҪңеҠӣгҖӮ
еӣҫиЎЁеҲ¶дҪң пјҡеӣҫиЎЁеҲ¶дҪңж—¶еёёиў«дәәд»¬еҝҪз•ҘпјҢдҪҶж— и®әдјҒдёҡеҶ…йғЁиҝҳжҳҜдјҒдёҡй—ҙдәӨжөҒеҗҲдҪңйғҪзҰ»дёҚејҖеӣҫиЎЁгҖӮзҺ°еңЁдёҖдәӣејҖеҸ‘иҖ…дҪҝз”ЁHTML/CSSе®ҢжҲҗеӣҫж ҮеҲ¶дҪңпјҢе…¶е®һеӨ§е®¶е®Ңе…ЁеҸҜд»Ҙз”ЁCanvasжқҘе®һзҺ°гҖӮеҪ“з„¶пјҢдҪҝз”ЁSVGпјҲеҸҜзј©ж”ҫзҹўйҮҸеӣҫеҪўпјүжқҘе®ҢжҲҗеӣҫиЎЁеҲ¶дҪңд№ҹжҳҜйқһеёёеҘҪзҡ„ж–№жі•гҖӮ
bannerе№ҝе‘Ҡ пјҡFlashжӣҫз»Ҹиҫүз…Ңзҡ„ж—¶д»ЈпјҢжҷәиғҪжүӢжңәиҝҳжңӘжӣҫеҮәзҺ°гҖӮзҺ°еңЁд»ҘеҸҠжңӘжқҘзҡ„жҷәиғҪжңәж—¶д»ЈпјҢHTML5жҠҖжңҜиғҪеӨҹеңЁbannerе№ҝе‘ҠдёҠеҸ‘жҢҘе·ЁеӨ§дҪңз”ЁпјҢз”ЁCanvasе®һзҺ°еҠЁжҖҒзҡ„е№ҝе‘Ҡж•ҲжһңеҶҚеҗҲйҖӮдёҚиҝҮгҖӮ

еӯ—дҪ“и®ҫи®Ў пјҡеҜ№дәҺеӯ—дҪ“зҡ„иҮӘе®ҡд№үжёІжҹ“е°Ҷе®Ңе…ЁеҸҜд»ҘеҹәдәҺWebпјҢдҪҝз”ЁHTML5жҠҖжңҜиҝӣиЎҢе®һзҺ°гҖӮ
еӣҫеҪўзј–иҫ‘еҷЁ пјҡеӣҫеҪўзј–иҫ‘еҷЁжңӘжқҘд№ҹи®ёиғҪеӨҹ100%еҹәдәҺWebе®һзҺ°гҖӮ
е…¶д»–еҸҜеөҢе…ҘзҪ‘з«ҷзҡ„еҶ…е®№ пјҡзұ»дјјеӣҫиЎЁгҖҒйҹійў‘гҖҒи§Ҷйў‘пјҢиҝҳжңүи®ёеӨҡе…ғзҙ иғҪеӨҹжӣҙеҘҪең°дёҺWebиһҚеҗҲпјҢе№¶дё”дёҚйңҖиҰҒд»»дҪ•жҸ’д»¶гҖӮеҸҜд»ҘеӨ§е®¶з»§з»ӯжҢ–жҺҳCanvasзҡ„жҪңеҠӣпјҢиҝҗз”ЁHTML5жҠҖжңҜеҲӣйҖ жӣҙеӨҡд»·еҖјгҖӮ

canvas зҡ„дёҖдәӣеә”з”ЁеңәжҷҜ

зІ’еӯҗеҠЁз”»е°ҸжёёжҲҸпјҡhttp://lab.hakim.se/trail/03/

еӣҫиЎЁеҲ¶дҪңпјҡhttp://echarts.baidu.com/gallery/editor.html?c=area-simple

зўҺзүҮејҸиҪ®ж’ӯеӣҫпјҡhttps://darylxyx.github.io/Demo/fragment/

еӣҫзүҮзј–иҫ‘пјҡ http://wow.techbrood.com/fiddle/2580

еҹәжң¬з”Ёжі•

иҰҒдҪҝз”Ё canvas е…ғзҙ пјҢе…Ҳи®ҫзҪ®е…¶ width е’Ң height еұһжҖ§гҖӮ

иҰҒеңЁз”»еёғдёҠз»ҳеӣҫпјҢйңҖиҰҒеҸ–еҫ—з»ҳеӣҫдёҠдёӢж–ҮпјҢйңҖиҰҒи°ғз”Ё getContextпјҲпјүж–№жі•е№¶дј е…ҘвҖң2dвҖқпјҢе°ұеҸҜд»ҘеҸ–еҫ—2DдёҠдёӢж–ҮеҜ№иұЎгҖӮ

2DдёҠдёӢж–Үзҡ„дёӨз§Қеҹәжң¬з»ҳеӣҫж“ҚдҪңпјҡеЎ«е……е’ҢжҸҸиҫ№пјҢеҸ–еҶідәҺдёӨдёӘеұһжҖ§пјҡ fillStyleе’ҢstrokeStyleгҖӮ

йўңиүІеҖјеҸҜд»ҘдҪҝз”Ё CSS дёӯжҢҮе®ҡйўңиүІеҖјзҡ„д»»дҪ•ж јејҸгҖӮ

з»ҳеҲ¶зҹ©еҪў

зҹ©еҪўжҳҜе”ҜдёҖдёҖз§ҚеҸҜд»ҘзӣҙжҺҘеңЁ2DдёҠдёӢж–Үз»ҳеҲ¶зҡ„еҪўзҠ¶гҖӮ

дёҺзҹ©еҪўжңүе…ізҡ„ж–№жі•пјҡfillRectпјҲпјү , strokeRectпјҲпјү , clearRectпјҲпјүгҖӮ

йңҖжҢҮе®ҡпјҲy,width,heightпјүиҝӣиЎҢз»ҳеҲ¶гҖӮ

з»ҳеҲ¶и·Ҝеҫ„

иҰҒз»ҳеҲ¶и·Ҝеҫ„пјҢе…Ҳи°ғз”ЁbeginPathпјҲпјүж–№жі•пјҢиЎЁзӨәиҰҒејҖе§Ӣз»ҳеҲ¶ж–°и·Ҝеҫ„гҖӮ

и°ғз”Ёз»ҳеҲ¶и·Ҝеҫ„зҡ„еҹәжң¬ж–№жі•пјҡ

arcпјҲx, y, radius, startAngle, endAngle, counterclockwiseпјү

moveToпјҲx, yпјү

lineToпјҲx, yпјү

rectпјҲx, y, width, heightпјү

quadraticCurveToпјҲcxпјҢxyпјҢxпјҢyпјү

bezierCurveToпјҲc1xпјҢc1yпјҢc2xпјҢc2yпјҢxпјҢyпјү

еҰӮжһңжғіиҰҒз»ҳеҲ¶дёҖжқЎиҝһжҺҘеҲ°и·Ҝеҫ„иө·зӮ№зҡ„зәҝжқЎпјҢеҸҜд»Ҙи°ғз”ЁclosePathпјҲпјүж–№жі•гҖӮ

еҰӮжһңжғіз”ЁfillStyleеЎ«е……и·Ҝеҫ„пјҢеҸҜд»Ҙи°ғз”ЁfillпјҲпјүж–№жі•иҝӣиЎҢеЎ«е……гҖӮ

еҰӮжһңжғіиҰҒз”ЁstrokeStyleеҜ№и·Ҝеҫ„иҝӣиЎҢжҸҸиҫ№пјҢеҸҜд»Ҙи°ғз”ЁstrokeпјҲпјү гҖӮ

beginPathпјҲпјүж–№жі•пјҡејҖе§Ӣз»ҳеҲ¶ж–°и·Ҝеҫ„пјҢдёәдәҶеҲҮж–ӯе’ҢдёҠдёҖдёӘи·Ҝеҫ„зҡ„иҒ”зі»гҖӮ

closePathпјҲпјүж–№жі•пјҡдёҖе®ҡиҰҒеңЁз»ҳеҲ¶д№ӢеүҚдҪҝз”ЁгҖӮ

з»ҳеҲ¶ж–Үжң¬

з»ҳеҲ¶ж–Үжң¬дё»иҰҒжңүдёӨдёӘж–№жі•пјҡfillTextпјҲпјүе’ҢstrokeTextпјҲпјү пјҢжҺҘеҸ—4дёӘеҸӮж•°пјҲж–Үжң¬пјҢxпјҢyпјҢжңҖеӨ§еғҸзҙ е®ҪеәҰпјүпјҢиҝҷдёӨдёӘж–№жі•йғҪд»ҘдёӢеҲ—дёүдёӘеұһжҖ§дёәеҹәзЎҖпјҡ

font пјҡиЎЁзӨәж–Үжң¬ж ·ејҸгҖҒеӨ§е°ҸеҸҠеӯ—дҪ“пјҢз”ЁCSSдёӯжҢҮе®ҡеӯ—дҪ“ж јејҸжқҘжҢҮе®ҡгҖӮ

textAlignпјҡиЎЁзӨәж–Үжң¬еҜ№йҪҗж–№ејҸгҖӮ

textBaselineпјҡиЎЁзӨәж–Үжң¬еҹәзәҝгҖӮ

еҸҳжҚў

rotateпјҲangleпјүпјҡеӣҙз»•еҺҹзӮ№ж—ӢиҪ¬еӣҫеғҸangleеј§еәҰгҖӮ

scaleпјҲscaleXпјҢscaleYпјүпјҡзј©ж”ҫеӣҫеғҸгҖӮ

translateпјҲxпјҢyпјүпјҡе°Ҷеқҗж ҮеҺҹзӮ№з§»еҠЁеҲ°пјҲxпјҢyпјү гҖӮ

йҳҙеҪұ

2DдёҠдёӢж–Үдјҡж №жҚ®дёӢеҲ—еҮ дёӘеұһжҖ§еҖјпјҢдёәеҪўзҠ¶жҲ–и·Ҝеҫ„з»ҳеҲ¶еҮәйҳҙеҪұгҖӮ

shadowColorпјҡз”ЁCSSйўңиүІж јејҸиЎЁзӨәзҡ„йҳҙеҪұйўңиүІгҖӮ

shadowOffsetXпјҡxиҪҙж–№еҗ‘еҒҸз§»йҮҸ

shadowOffsetYпјҡyиҪҙж–№еҗ‘еҒҸз§»йҮҸ

shadowBlurпјҡжЁЎзіҠеғҸзҙ ж•°

жёҗеҸҳ

еҲӣе»әжёҗеҸҳеҜ№иұЎпјҢcreateLinearGradientпјҲпјүпјҡжҺҘж”¶4дёӘеҸӮж•°пјҡиө·зӮ№е’Ңз»ҲзӮ№зҡ„xгҖҒyеқҗж ҮеҖјгҖӮ

еҲӣе»әеҫ„еҗ‘жёҗеҸҳпјҢcreateRadialGradientпјҲпјүпјҡжҺҘж”¶6дёӘеҸӮж•°пјҡиө·зӮ№еңҶе’Ңз»ҲзӮ№еңҶзҡ„еңҶеҝғгҖҒеҚҠеҫ„

жҢҮе®ҡиүІж ҮпјҢaddColorStopпјҲпјүпјҡжҺҘж”¶2дёӘеҸӮж•°пјҡиүІж ҮдҪҚзҪ®пјҲ0 ~ 1пјүпјҢиүІж ҮйўңиүІеҖј

saveе’Ңrestore

ж— и®әжҳҜжү§иЎҢеҸҳжҚўиҝҳжҳҜfillStyleгҖҒstrokeStyleзӯүеұһжҖ§пјҢйғҪдјҡеңЁеҪ“еүҚдёҠдёӢж–ҮдёӯдёҖзӣҙжңүж•ҲпјҢйҷӨйқһеҶҚеҜ№дёҠдёӢж–ҮиҝӣиЎҢдҝ®ж”№гҖӮеҰӮжһңе°ҶжқҘиҝҳиҰҒиҝ”еӣһжҹҗз»„еұһжҖ§е’ҢеҸҳжҚўзҡ„з»„еҗҲпјҢеҸҜд»Ҙи°ғз”ЁsaveпјҲпјүзҡ„ж–№жі•гҖӮsaveпјҲпјүж–№жі•дјҡе°ҶеҪ“еүҚжүҖжңүи®ҫзҪ®еӯҳе…ҘдёҖдёӘж Ҳз»“жһ„гҖӮ

еҜ№дёҠдёӢж–ҮиҝӣиЎҢдҝ®ж”№еҗҺпјҢеҰӮжһңжғіиҰҒжҒўеӨҚд№ӢеүҚзҡ„зҠ¶жҖҒж—¶пјҢеҸҜд»Ҙи°ғз”ЁrestoreпјҲпјүж–№жі•пјҢеңЁдҝқеӯҳи®ҫзҪ®зҡ„ж Ҳз»“жһ„дёӯеҗ‘еүҚиҝ”еӣһдёҖзә§пјҢжҒўеӨҚд№ӢеүҚзҡ„зҠ¶жҖҒгҖӮ

saveпјҲпјүж–№жі•еҸҜд»Ҙиҝһз»ӯдҪҝз”ЁдҝқеӯҳжӣҙеӨҡи®ҫзҪ®еҲ°ж ҲдёӯпјҢrestoreпјҲпјүж–№жі•иҝһз»ӯи°ғз”ЁеҸҜд»ҘдёҖзә§дёҖзә§иҝ”еӣһд№ӢеүҚзҡ„и®ҫзҪ®гҖӮ

saveпјҲпјүж–№жі•дҝқеӯҳзҡ„еҸӘжҳҜеҜ№з»ҳеӣҫдёҠдёӢж–Үзҡ„и®ҫзҪ®е’ҢеҸҳжҚўпјҢдёҚдјҡдҝқеӯҳз»ҳеӣҫдёҠдёӢж–Үзҡ„еҶ…е®№гҖӮ

е®һдҫӢDEMO1пјҡ еҹәдәҺ canvas зҡ„з®Җжҳ“ж—¶й’ҹ

з»ҳеҲ¶еӣҫеғҸ

2DдёҠдёӢж–ҮеҶ…зҪ®дәҶеҜ№еҜ№иұЎзҡ„ж”ҜжҢҒгҖӮеҸҜд»ҘдҪҝз”ЁdrawImageпјҲпјүж–№жі•жҠҠеӣҫзүҮз»ҳеҲ¶еҲ°з”»еёғдёҠ

drawImageпјҲпјүпјҡ

2дёӘеҸӮж•°пјҡimgе…ғзҙ пјҢеқҗж Ү

5дёӘеҸӮж•°пјҡimgе…ғзҙ пјҢеқҗж ҮпјҢе®Ҫй«ҳ

9дёӘеҸӮж•°пјҡimgе…ғзҙ пјҢжәҗеӣҫеғҸеқҗж ҮпјҢжәҗеӣҫеғҸе®Ҫй«ҳпјҢзӣ®ж ҮеӣҫеғҸеқҗж ҮпјҢзӣ®ж ҮеӣҫеғҸе®Ҫй«ҳ

дҪҝз”ЁеӣҫеғҸж•°жҚ®

2DдёҠдёӢж–ҮеҸҜд»ҘйҖҡиҝҮgetImageDataпјҲпјүеҸ–еҫ—еҺҹе§ӢеӣҫеғҸж•°жҚ®гҖӮ

getImageDataпјҲпјүпјҡжҺҘж”¶4дёӘеҸӮж•°пјҡеқҗж ҮпјҢе®Ҫй«ҳеәҰпјҢиҝ”еӣһдёҖдёӘе®һдҫӢеҜ№иұЎ

жҜҸдёҖдёӘе®һдҫӢеҜ№иұЎйғҪжңүдёүдёӘеұһжҖ§пјҡwidthпјҢheightе’Ңdata

dataеұһжҖ§дҝқеӯҳзқҖеӣҫеғҸдёӯжҜҸдёҖдёӘеғҸзҙ зҡ„ж•°жҚ®гҖӮ

жҜҸдёҖдёӘеғҸзҙ з”Ё4 дёӘе…ғзҙ жқҘдҝқеӯҳпјҡзәўгҖҒз»ҝгҖҒи“қе’ҢйҖҸжҳҺеәҰеҖјгҖӮ

е®һдҫӢDEMO2пјҡ еӣҫеғҸи§ЈжһҗзІ’еӯҗ
